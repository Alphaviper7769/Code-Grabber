216. Combination Sum III
Medium

Find all valid combinations of k numbers that sum up to n such that the following conditions are true:

  • Only numbers 1 through 9 are used.
  • Each number is used at most once.

Return a list of all possible valid combinations. The list must not contain the same combination twice, and the combinations may be returned in any order.

 

Example 1:

Input: k = 3, n = 7
Output: [[1,2,4]]
Explanation:
1 + 2 + 4 = 7
There are no other valid combinations.

Example 2:

Input: k = 3, n = 9
Output: [[1,2,6],[1,3,5],[2,3,4]]
Explanation:
1 + 2 + 6 = 9
1 + 3 + 5 = 9
2 + 3 + 4 = 9
There are no other valid combinations.

Example 3:

Input: k = 4, n = 1
Output: []
Explanation: There are no valid combinations.
Using 4 different numbers in the range [1,9], the smallest sum we can get is 1+2+3+4 = 10 and since 10 > 1, there are no valid combination.

 

Constraints:

  • 2 <= k <= 9
  • 1 <= n <= 60

If you're stuck, perhaps consider a DP-style approach:
In every combination sum, either the first item (let's call it t) is included, or it isn't.
If it is included, we can k-- and n-=t, and recurse.
If it's excluded, we can remove t from our potential list of numbers and recurse.
Finally, the full list is the sum of combinations where t is included and combinations where t is excluded.

For a further hint, here's a snippet:
q = whatsleft.pop()
return [comb + [q] for comb in func(k - 1, n - q, whatsleft.copy())] + func(k, n, whatsleft.copy())
